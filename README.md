## Descripción

**IslaControl Solutions** es una plataforma de gestión empresarial que incluye:

- Punto de Venta (POS)
- Control de inventario y productos
- Gestión de clientes
- Control de caja y reportes
- Asistente de inteligencia artificial
- Sistema de suscripciones

## Tecnologías

- **Backend:** Laravel (PHP)
- **Frontend:** Blade + Tailwind CSS
- **Base de datos:** MySQL
- **IA:** Groq API
